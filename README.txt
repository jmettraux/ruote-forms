
= ruote-forms

json to html form and back, a javascript library.

a part of ruote-web2 (http://github.com/jmettraux/ruote-web2)


== playing with it

  sudo gem install rack mongrel
  git clone git://github.com/jmettraux/ruote-forms.git
  ruby start.rb

point browser to

  http://localhost:4567
